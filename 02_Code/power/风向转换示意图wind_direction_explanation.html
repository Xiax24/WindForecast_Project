🧭 风向角度与三角函数分量关系

N (0°)
S (180°)
E (90°)
W (270°)

实时计算值

角度 (θ): 45°
cos(θ) - 东西分量: 0.707
sin(θ) - 南北分量: 0.707
方向描述: 东北

🔍 为什么要进行这种转换？

问题：风向是圆形数据，0°和360°实际上是同一个方向（北方），但数值上相差很大，直接计算相关性会产生错误结果。

解决方案：将气象角度转换为笛卡尔坐标系：

  • 数学转换：气象角度θ → 数学角度 (90° - θ)
  • cos(90° - θ) → X轴分量 → 东西方向（正值向东，负值向西）
  • sin(90° - θ) → Y轴分量 → 南北方向（正值向北，负值向南）

📐 典型风向示例

北风 (0°)

cos(90°) = 0.0 (无东西)

sin(90°) = 1.0 (北)

东风 (90°)

cos(0°) = 1.0 (东)

sin(0°) = 0.0 (无南北)

南风 (180°)

cos(270°) = 0.0 (无东西)

sin(270°) = -1.0 (南)

西风 (270°)

cos(180°) = -1.0 (西)

sin(180°) = 0.0 (无南北)

💡 相关性分析的优势

转换后的优势：

  • 连续性：消除了0°/360°边界问题
  • 线性相关：可以使用标准的Pearson/Spearman相关性
  • 物理意义：分别代表东西和南北风力分量
  • 标准化处理：sin和cos值都在[-1,1]范围内
